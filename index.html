1.1 Conceptos básicos de estadística
  • Definición de estadística: Ciencia que se encarga de recopilar, organizar, analizar e interpretar datos para obtener conclusiones.
  • Teoría de decisión: Proceso de toma de decisiones basado en el análisis de datos y probabilidades.
  • Población: Conjunto completo de elementos que comparten características de interés.
  • Muestra aleatoria: Subconjunto de la población seleccionado al azar para su estudio.
  • Parámetros aleatorios: Características de una población que se modelan como variables aleatorias.
1.2 Descripción de datos
  • Datos agrupados y no agrupados: Datos presentados de forma individual o en intervalos.
  • Frecuencia de clase: Número de datos que caen en un determinado intervalo o clase.
  • Frecuencia relativa: Proporción de datos en una clase con respecto al total.
  • Punto medio: Valor central de un intervalo de clase.
  • Límites: Valores máximo y mínimo de un intervalo de clase.
1.3 Medidas de tendencia central y dispersión
  • Media aritmética: Valor promedio de un conjunto de datos.
  • Media geométrica: Media de un conjunto de datos expresados como productos.
  • Media ponderada: Media que considera la importancia relativa de cada valor.
  • Mediana: Valor central de un conjunto de datos ordenados.
  • Moda: Valor que aparece con mayor frecuencia en un conjunto de datos.
  • Varianza: Medida de dispersión que indica la dispersión de los datos respecto a la media.
  • Desviación estándar: Raíz cuadrada de la varianza, mide la dispersión de los datos.
  • Desviación media: Promedio de las diferencias absolutas entre cada valor y la media.
  • Desviación mediana: Promedio de las diferencias absolutas entre cada valor y la mediana.
  • Rango: Diferencia entre el valor máximo y mínimo de un conjunto de datos.
